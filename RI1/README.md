# RI1 — Vingadores vs Thanos

Projeto em JavaScript que cria objetos representando personagens do universo Marvel (Vingadores e Thanos) e realiza comparações entre seus atributos.

## Objetivos

- Criar objetos literais para cada super-herói e para o Thanos
- Cada objeto contém os atributos: `nome`, `codinome`, `armaPrincipal`, `armaSecundaria`, `velocidade`, `forca`, `resistencia` e o método `descricao()`
- Todos os personagens são armazenados em um Array
- Estruturas de repetição (`for`) percorrem o Array e comparam os atributos de cada herói com os do Thanos
- O programa imprime a descrição de cada personagem e aponta quem vence em cada atributo numérico

## Personagens

| Codinome        | Nome Real        |
|-----------------|------------------|
| Homem de Ferro  | Tony Stark       |
| Capitão América | Steve Rogers     |
| Thor            | Thor Odinson     |
| Viúva Negra     | Natasha Romanoff |
| Hulk            | Bruce Banner     |
| O Titã Louco    | Thanos           |

## Como executar

```bash
node index.js
```

## Tecnologias
